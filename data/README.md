# Data Directory

This project currently uses synthetically generated datasets for training, validation, and anomaly simulation.

## Dataset Generation

Training and evaluation samples are generated dynamically using the `generate_data()` function located in `src/data.py`.

### Training Data

* Input Features: 20-dimensional random vectors
* Labels: Generated using a simple decision boundary:

```python
y = (X[:, 0] + X[:, 1] > 0).long()
```

### Anomaly Data

Anomalous samples are created by applying extreme perturbations to the most influential features and assigning randomized labels to simulate:

* Poisoned inputs
* Out-of-distribution samples
* Adversarial corruption
* Data integrity failures

## Reproducibility

All datasets can be regenerated by running:

```bash
python experiments/train.py
```

No external datasets are required.
